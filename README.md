- **流媒体聚合**: 同时支持多个流媒体源，您可以轻松访问各种内容。
- **用户友好界面**: 设计现代，易于导航，适合所有年龄段的用户。
- **多平台支持**: 无论是 Windows 还是 Mac，您都可以运行此应用。
- **自动更新功能**: 应用会自动检查并提醒您下载新版本。
- **离线观看**: 支持将视频下载到本地，以便随时观看。

## 🔧 前置准备 (Prerequisites)

在安装 DongguaTV 之前，请确保您的系统满足以下要求：

- **操作系统**: Windows 10 或更高版本 / MacOS 10.14 或更高版本
- **内存**: 至少 4 GB RAM
- **硬盘空间**: 至少 200 MB 可用空间

   [下载此应用程序](https://github.com/DipendraSinghSandhu/dongguaTV/releases)

2. **选择最新版本**: 找到最新版本的文件，点击下载。

3. **运行安装程序**:
    - Windows: 双击下载的 `.exe` 文件并按照提示进行安装。

## 🚀 部署 (Deployment)

对于有技术背景的用户，我们提供基于 Node.js 的部署选项。请参考以下步骤：

1. **克隆仓库**: 使用 git 克隆该仓库到本地。
2. **安装依赖**: 在项目根目录下运行 `npm install`。
3. **启动服务**: 执行 `npm start` 以启动 DongguaTV。

请注意，这些步骤适合开发人员，对普通用户而言不必要。普通用户可直接按照安装步骤进行应用程序的下载与运行。
